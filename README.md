## Markdowm
**MarkDown是什么**
- Markdown是一种可以使用普通文本编辑器编写的标记语言，用于排版

**MarkDown的特点**
- 简单标记符完成排版，所写即所见，让你专注于文字而不是排版
- 纯文本，所以兼容性极强，可以用所有文本编辑器打开
- 格式转换方便，Markdown文本可以轻松转换为 html、pdf等
- Markdown 的标记语法有极好的可读性

**MarkDown的用途**
- 用于编写说明文档
- 我们还可以快速将Markdown转化为演讲PPT、Word产品文档
- 写博客
